# Pixel Sprite Cuter

一个面向 `macOS` 的像素精灵切分与重排工具。

它的目标很直接：把一张纯色或近似纯色背景上的零散像素精灵自动识别出来，再按你指定的尺寸分组，导出成标准精灵表。

## 现在支持什么

- 导入 `PNG / JPG / WEBP` 图片
- 自动估计背景色并识别独立精灵
- 支持背景容差、最小面积、最小宽高调节
- 支持单选分组、批量选择、批量分组、批量忽略
- 同一张图重新识别时，尽量保留已有分组和忽略结果
- 每个分组按固定列数导出透明背景 PNG 精灵表
- 导出时支持两种边缘处理模式：
  - `标准模式`
  - `强去边（黑描边素材）`
- 导出重采样使用最近邻，适合像素风素材

## 适合什么素材

当前最适合：

- 白底或近白底的像素角色/物件图
- 带黑描边的像素素材
- 精灵彼此分散、不重叠的杂图

当前不适合：

- 复杂摄影背景
- 大面积重叠或粘连的精灵
- 需要语义级抠图的插画素材

## 导出模式说明

### 标准模式

导出前做基础边缘净化，适合一般纯色背景素材。

### 强去边（黑描边素材）

只在导出时生效，不影响左侧识别预览。

这个模式会：

1. 先分析精灵外轮廓的 1-3 圈像素
2. 对被白底污染的浅灰/浅褐边缘做颜色去污染
3. 再对最外层残留 halo 做轻裁切
4. 最后再做最近邻重采样和排版

如果你的素材是白底、黑描边、在深色背景上容易出现明显白边，优先试这个模式。

## 本地开发

### 环境

- Node.js 24+
- npm 11+

### 安装依赖

```bash
npm install
```

### 启动开发模式

```bash
npm run dev
```

### 运行测试

```bash
npm test
```

### 构建

```bash
npm run build
```

### 打包 macOS 应用

```bash
npm run package
```

打包后的产物会出现在 `release/` 目录。

## 项目结构

```text
src/
  core/       图像识别、边缘清理、导出重采样逻辑
  main/       Electron 主进程
  preload/    Electron bridge
  renderer/   React 界面、交互与状态
  types/      Electron 和共享类型
tests/
  core.test.ts
```
